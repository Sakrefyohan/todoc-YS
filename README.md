# Todoc - Yohan Sakref

This repository contains a mini-application for the Project 5 from OpenClassrooms course. Modified by me.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.


## Purpose of the project - Todoc
------------------------------
The objective of this project is to make data persistent by being stored in a database.


Features created :
    - add a task to the database
    - delete a task from the database
    - retrieve the list of tasks contained in the database
